## Workout Log

Workout Log is a Next.js pet project that helps you create custom training plans and track reps, sets, and progress. Plan your workouts ahead, log what you've done, and see how you improve over time.

## Tech Stack

IndexedDB – For client-side data persistence (no backend required)
Zustand – Lightweight and performant state management
Next.js – Used for learning purposes (though plain React would have sufficed)

## Getting Started

Clone the repo:

```bash
git clone <your-repo-url>
cd workout-log
```

Install dependencies:

```bash
yarn install
# or
npm i
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
